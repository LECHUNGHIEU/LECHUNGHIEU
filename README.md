## 🚀 Featured Projects

### 🖥️ 32-Bit Pipelined RISC-V RV32I Processor

- Designed and implemented a 32-bit RV32I processor using a modular RTL architecture.
- Developed a 5-stage pipelined datapath (IF, ID, EX, MEM, WB).
- Implemented Hazard Detection and Forwarding Units.
- Developed ALU, Register File, Control Unit, and Memory Interface modules.
- Verified functionality using Cadence Xcelium.
- Validated implementation on DE2-115 FPGA.

### 🔐 CRYSTALS-Dilithium NTT/INTT Accelerator

- Designed a 256-point NTT/INTT accelerator for polynomial multiplication.
- Implemented Butterfly Processing Units and Montgomery Reduction.
- Developed Address Generators, Twiddle-Factor ROM Controllers, and FSM-based Control Logic.
- Designed Dual-Port Memory Architecture for parallel data access.
- Integrated NTT, Point-Wise Multiplication, and INTT modules into a complete accelerator.

- Implemented Key Expansion, Cipher Engine, Counter Generator, and Control Unit.
- Developed FSM-based Control Architecture.
- Verified functionality through RTL simulation and test vectors.
- Evaluated timing performance and FPGA resource utilization.
